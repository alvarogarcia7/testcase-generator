# Sample Data Directory

This directory contains sample data files for testing and validation purposes.

## Files

- `testcase_results_container_sample.yml` - Sample test case results container data

## Purpose

Sample data files in this directory are used for:
- Schema validation testing
- Documentation examples
- Integration testing
- Reference implementations

## Notes

Sample data files were previously scattered across various subdirectories and have been consolidated here for better organization and easier maintenance.

All sample files should validate against their corresponding schema files located in `schemas/tcms/` or other schema directories.
