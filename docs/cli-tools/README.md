# CLI Tools Reference

Command-line tools for test management and execution.

## Core Tools

### Validation

- **[validate-yaml](validate-yaml.md)** - YAML validation with watch mode
  - Single or multi-file validation
  - Continuous monitoring
  - Schema enforcement

## Quick Reference

| Tool | Purpose | Example |
|------|---------|---------|
| `validate-yaml` | Validate YAML files | `validate-yaml testcase.yml --schema schema.json --watch` |
| `editor` | Create/edit test cases | `editor complete --output testcases/test.yml` |
