# Features

Advanced features for test management and automation.

## Core Features

### BDD & Natural Language

- **[BDD Initial Conditions](bdd-initial-conditions.md)** - Human-readable conditions that convert to bash commands
  - 23 built-in patterns
  - Custom pattern support

### Variables & Data Flow

- **[Variable Passing](variable-passing.md)** - Capture and pass data between steps
  - Regex capture
  - Variable substitution
  - Scope management

### User Experience

- **[Recovery](recovery.md)** - Automatic progress saving
  - Save state after each operation
  - Resume interrupted workflows
  - Error tracking

## Getting Started

1. Start with [BDD Conditions](bdd-initial-conditions.md) for readable conditions
2. Learn [Variable Passing](variable-passing.md) for dynamic data
3. Explore [Recovery](recovery.md) for workflow resilience

For examples, see the [Examples](../examples/) section.
